# LeeAIex 🚀  
AI-powered coding assistant for LeetCode

---

## ✨ Vision  
**LeeAIex** empowers developers to learn problem-solving step-by-step.  
It’s like having a **smart tutor inside your browser**, giving hints and explanations **without giving away the solution directly**.

---

## 💡 What is LeeAIex?  
**LeeAIex** is a Chrome extension + AI backend that provides:  
- 🎯 Step-by-step coding guidance  
- ⚡ Smart hints for LeetCode problems  
- 🤖 Minimal, intuitive interface  
- 🛠️ Works directly in-browser with no extra setup

## 🚀 Features  
✅ Floating **Ask LeeAIex** button  
✅ Interactive chat window  
✅ Hint system: 3 guided hints before full input unlocked  
✅ Custom input box for questions  
✅ Step-by-step explanation from AI backend  
✅ Auto-scroll chat with styled messages  

---

## 🛠️ How It Works

### Frontend Flow
1. Install the Chrome extension  
2. Open a LeetCode problem  
3. Click the **Ask LeeAIex** button  
4. View hints or type your own question  
5. AI responses appear in the chat window, styled and scrollable
